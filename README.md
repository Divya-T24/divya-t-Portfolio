- `script.js` - Interactive JavaScript

- `resume.pdf` - Placeholder PDF

## How to Generate Resume PDF:
1. Open `resume.html` in your browser
2. Press Ctrl+P (or Cmd+P on Mac) to print
3. Select "Save as PDF" in the print dialog
4. Save as "Divya_T_Resume.pdf"
